# NextPat
Human-AI Collaboration for Design Ideation and Patent Drafting

NextPat is an intelligent system built upon the Human-AI Co-creation (HAIC) theoretical model. It bridges the gap between raw innovative inspiration and rigorous legal documentation, specifically designed for Patent-Oriented Inventive Design (POID).

The system is currently deployed and operational on ScholarMate, the largest professional research social networking platform in China.
